### 📖 项目介绍 📖

这是我的第一个 Speed Code 视频项目，通过此项目可以学习到 `Container、AnimatedSwitcher、FadeTransition、ScaleTransition` 等组件的基础用法，最终我们将构建一个灵动的录音状态切换按钮

> 需要说明的是按钮全部是用代码编写的，`圆环` 和 `圆形` 都是 `Container` 的属性效果，具体可以看文章或视频
